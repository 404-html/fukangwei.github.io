json模块

发表于 2018-12-28

  JSON(JavaScript Object Notation)是一种轻量级的数据交换格式，它是基于ECMAScript的一个子集。Python 3中可以使用json模块来对JSON数据进行编解码，它包含了两个函数：

  • json.dumps：对数据进行编码。
  • json.loads：对数据进行解码。

在json的编解码过程中，python的原始类型与json类型会相互转换。Python编码为JSON类型转换对应表如下所示：

Python JSON
dict object
list, tuple array
str string
int, float, int- & float-derived Enums number
True true
False false
None null

JSON解码为Python类型转换对应表如下所示：

JSON Python
object dict
array list
string str
number (int) int
number (real) float
true True
false False
null None
